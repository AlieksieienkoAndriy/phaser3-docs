boot()

Description
If this is a Scene Plugin (i.e. installed into a Scene) then this method is called when the Scene boots. By this point the plugin properties `scene` and `systems` will have already been set. In here you can listen for Scene events and set-up whatever you need for this plugin to run.

stop()

Description
Called by the PluginManager when this plugin is stopped. The game code has requested that your plugin stop doing whatever it does. It is now considered as 'inactive' by the PluginManager. Handle that process here (i.e. stop listening for events, etc) If the plugin is started again then `BasePlugin.start` will be called again.
