new TextureManager( game )

Description
Textures are managed by the global TextureManager. This is a singleton class that is responsible for creating and delivering Textures and their corresponding Frames to Game Objects. Sprites and other Game Objects get the texture data they need from the TextureManager. Access it via `scene.textures`.
